# Product Roadmap

This document outlines the planned features and enhancements for Calculator Basics.

##  Current Status: Version 2.0

**Released**: Core functionality with scientific features, themes, memory, and accessibility.

##  In Progress

### Short-term (Next Release)

- [ ] **Unit Conversions Module**
  - Temperature (Celsius, Fahrenheit, Kelvin)
  - Length (Meters, Feet, Yards, Miles, etc.)
  - Weight (Kilograms, Pounds, Ounces, etc.)
  - Time (Hours, Minutes, Seconds, etc.)
  - Separate conversion mode accessible from scientific mode

- [ ] **Enhanced Expression Parser**
  - Full PEMDAS/BODMAS implementation
  - Support for complex nested expressions
  - Shunting-yard algorithm for proper parsing
  - Support for mixed operations in single expression

- [ ] **Additional Constants**
  - Speed of light
  - Gravitational constant
  - Planck constant
  - Avogadro's number
  - Golden ratio (Ï†)

- [ ] **Display Enhancements**
  - Configurable precision (3-15 decimal places)
  - Thousand separators
  - Localization support (different number formats)
