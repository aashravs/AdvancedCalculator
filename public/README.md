# Public Assets

This folder contains public static assets.

## Demo GIF

Add a demo GIF or screenshot showcasing the calculator to this folder as `demo.gif`.

The recommended size is 600x400 pixels or similar aspect ratio.

## How to Create a Demo

1. Record a screen capture of the calculator in action
2. Convert to GIF format (tools: LICEcap, Kap, or online converters)
3. Name it `demo.gif`
4. Place in this folder

The README will automatically display this demo in the main documentation.
